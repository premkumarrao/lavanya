1. For git config user name:
git config --global user.name "Your name",

2. For git config email name:
git config --global user.email "youremail@domain.com"

3. For git clone (https):
git clone (url) 

4. For git add files:
git add (file name with extension)

5. For git status:
git status

6. For git commit;
git commit -m "(commited message)" file name
